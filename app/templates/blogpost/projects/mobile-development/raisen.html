Raisen

AI-Powered Itinerary & Collage Generator

Raisen is a personal project I developed to simplify travel planning and offer creative visual content to users. Built with Swift and powered by Firebase for data management, Raisen focuses on two core functionalities: generating personalized travel schedules using the OpenAI API and creating city-inspired collages via DALL·E. Like my other work, the app supports Google, Apple, or email sign-in to offer a seamless login experience.

Similar to my other app projects, Firebase Authentication handles user logins in Raisen, allowing quick and secure access through familiar platforms. I wanted to keep the onboarding as simple as possible while maintaining strong security measures. As a solo developer, this approach let me focus on the core features—analyzing user input and integrating AI solutions—without rewriting login and database logic.

At the heart of Raisen is an itinerary generation tool. Upon entering a city name and specifying how many days the user will stay, the app communicates with the OpenAI API to create a personalized travel schedule. This schedule includes recommended tourist spots, optimal routes to visit them, an estimate of how long users might spend at each site, and even images that showcase some highlights of the city.

To keep everything organized, I used Firestore (a part of Firebase) to store user preferences—like which city they're interested in, how many days they'll be there, and any personal constraints or preferences. This data structure makes it easy to revisit or update a previously generated schedule. The UI was designed to feel minimal and intuitive, so users can quickly see the most critical information about their trip without digging through cluttered menus.

Beyond itinerary planning, Raisen features a collage generator that uses the DALL·E API to produce unique background images based on the user’s chosen city. By simply entering a city name, the app creates an image collage that can be used as a device wallpaper or shared on social media. This feature was particularly fun to develop, as it merges creative AI capabilities with user-generated inputs, providing a delightful, personalized result.

In the Settings Screen, users can update their login method, adjust notification preferences for future trip alerts, and of course log out or revoke access. Through Firebase, all user data remains secure, while providing a convenient way to sync preferences and maintain session state across sessions.

As a solo developer on this project, I handled everything from UI/UX design to API integrations. It has been a great learning experience, combining open-ended AI interactions with structured app components. Looking ahead, I plan to refine the routing suggestions by incorporating real-time data—such as traffic or local events—and expand the DALL·E collage feature for more creative output variations.

In essence, Raisen merges travel planning with AI-driven image generation, presenting a fresh approach to both functional and aesthetic needs of travelers. Whether you’re looking for an in-depth itinerary or a visually stunning backdrop reflecting your chosen destination, Raisen aims to deliver an engaging, user-centric experience. Stay tuned for ongoing feature enhancements as I continue to explore new possibilities with AI and mobile development.
